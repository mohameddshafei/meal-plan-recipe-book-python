# Meal Plan Recipe Book

## Overview
This is a Python object-oriented programming project that allows users to create recipes, organize them into meal plans, save meal plans to a text file, load saved meal plans, and search for meal plans by name.

The project demonstrates basic OOP concepts such as classes, objects, attributes, methods, lists, file handling, and simple searching.

## Features
- Create recipes with a name, ingredients, preparation time, and calories
- Add recipes to meal plans
- Save meal plans to a text file
- Load meal plan information from a text file
- Store multiple meal plans in a recipe book
- Display available meal plans
- Search for a meal plan by name

## Classes Used

### Recipe
Stores recipe information:
- Recipe name
- Ingredients
- Preparation time
- Calories

### MealPlan
Stores a group of recipes and includes methods to:
- Add recipes
- Save meal plans to a file
- Load meal plans from a file

### RecipeBook
Stores multiple meal plans and includes methods to:
- Add meal plans
- Display all meal plans
- Search for a meal plan by name
